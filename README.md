Global Business Performance Ledger (2019-2023)
Project Overview
This repository contains a comprehensive business intelligence solution designed to track global sales performance, profitability, and product trends. The project demonstrates a full data lifecycle: from raw CSV ingestion and Power Query (M) transformation to data visualization.

Technical Stack
Business Intelligence: Power BI Desktop

Data Engineering: Power Query (M Language)

Data Visualisation: Power BI Desktop

Data Source: Flat-file CSV (raw_sales_data.csv)
